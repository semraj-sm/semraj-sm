## 🚀 Professional Summary

Senior .NET Engineer with **5+ years** of experience designing, developing, and deploying scalable enterprise applications using **C#**, **.NET Core**, **ASP.NET Core**, **React**, **SQL Server**, and **Microsoft Azure**.

I specialize in architecting **cloud-native microservices**, building **RESTful APIs**, optimizing distributed systems, and delivering end-to-end solutions in **Agile** environments.

I bring strong expertise across Azure services including **App Services**, **Azure Functions**, **Cosmos DB**, **Event Hub**, and CI/CD pipelines through **Azure DevOps** and **GitHub Actions**. I have a proven track record in leading technical implementations, building POCs, troubleshooting production issues, and driving performance optimization.

I enjoy modern engineering practices around:
- Event-driven architecture
- Secure auth (JWT/OAuth2/RBAC)
- Database optimization
- Automated deployments
- AI-assisted development workflows
